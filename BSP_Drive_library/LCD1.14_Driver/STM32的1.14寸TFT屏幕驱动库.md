# STM32的1.14寸TFT屏幕驱动库

包括功能有驱动1.14寸TFT屏幕

### 使用教程：

1、keil导入tftlcd.c、tftlcd.h和tft_font.h文件；

2、并在tftlcd.h中进行初始化，包括io的选取、spi的选取（为硬件spi），更多设置可看tftlcd.h；

3、主函数里使用要先初始化并清屏

```c
/* USER CODE BEGIN 2 */
	LCD_Init();
	LCD_Clear(BLACK);
	LCD_ShowString(32, 0, 240, 16, 16, "Boring_TECH");
```
