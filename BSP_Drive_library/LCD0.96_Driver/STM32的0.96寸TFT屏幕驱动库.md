# STM32的0.96寸TFT屏幕驱动库

包括功能有驱动0.96寸TFT屏幕

### 使用教程：

1、keil导入tftlcd.c、tftlcd.h和tft_font.h文件；

2、并在tftlcd.h中进行初始化，包括io的选取、spi的选取（为硬件spi），更多设置可看tftlcd.h；

3、在cubemx里配置spi时需要选择：

（1）Mode选Half-Duplex Master

（2）Baud Rate不能太高

~~（3）Clock Polarity（CPOL）选High~~

~~（4）Clock Phase（CPHA）选2Edge~~

*注：0.96寸LCD屏和1.14寸的spi协议不一样*

4、主函数里使用要先初始化并清屏

```c
/* USER CODE BEGIN 2 */
	LCD_Init();
	LCD_Clear(BLACK);
	LCD_ShowString(32, 0, 160, 16, 16, "Boring_TECH");
```
